# 总体架构

```text
FSR等传感器
  ↓
ESP32-S3：采集→滤波→规则/模型推理
  ├─BLE→小程序实时页面
  ├─HTTP/Wi-Fi→FastAPI后端
  └─GPIO→振动、蜂鸣器、台灯
                    ↓
              数据库与统计
              ├─Web管理端
              └─小程序历史页面
```

当前第一阶段优先打通HTTP链路；BLE接口保留，等HTTP稳定后接入。

上传频率建议：
- 本地采样：100～200ms；
- BLE实时：200ms；
- 云端遥测：5s或姿态变化时立即上传。
